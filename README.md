# AI-GROQ-Personality


this is a chatbot connectede to GROQ API



TO RUN:

# create a venv: 
	pour mac/linux: python3 -m venv env
	pour windows : python -m venv env

# activate venv:
	pour mac/linux: source env/bin/activate
	pour windows: env\Scripts\activate


# install libraries
pip install -r requirements.txt


# create .env file
    -create the .env file in the same folder as this project

    -after creating the .env file paste this:
    """
    GROQ_API_KEY="....clef api ici...."
    """

# execute the code

    python main.py
